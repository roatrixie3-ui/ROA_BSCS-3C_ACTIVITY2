# ROA_BSCS-3C_ACTIVITY2
PORTFOLIO MULTIPAGE APP

 This is a personal portfolio application built using Streamlit.
 It features multiple pages to showcase information about me, my skills, projects, and contact details.
 
 Pages Included:
 - Home
 - About
 - Skills
 - Contact
 - Projects
   
 Technologies Used:
 - Python
 - Streamlit
 - Pillow
 - 
How to Run Locally:
 1. Clone the repository:
    git clone https://github.com/roatrixie3-ui/ROA_BSCS-3C_ACTIVITY2.git
 2. Navigate to the project directory:
    cd ROA_BSCS-3C_ACTIVITY2
 3. Install the required libraries:
    pip install -r requirements.txt
 4. Run the Streamlit app:
    streamlit run ROA_BSCS3C_ACT2/1_Home.py (Adjust the path if your main file is in a different location)
